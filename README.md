# Awesome-Chain-of-Thought
## Top AI Models Using Chain-of-Thought (CoT)

Chain-of-Thought (CoT) prompting has evolved from a prompt engineering technique into a built-in architecture for top-tier AI models. These models improve complex problem-solving by breaking tasks down into logical, step-by-step intermediate reasoning steps before delivering a final answer.

## Leading Proprietary Models

| Model | Key Features | Year | Paper / Resource Link |
| :--- | :--- | :--- | :--- |
| [**OpenAI (o-series)**](./docs/openai-o-series.md) | Models like o1 and o3-mini are trained with reinforcement learning to generate internal CoT tokens for math, coding, and logic. | 2024 | [OpenAI o1 System Card](https://arxiv.org/abs/2412.16720) |
| [**Google Gemini**](./docs/google-gemini.md) | Advanced versions feature built-in **"Deep Think"** and **"Think Mode"** for reasoning through multi-step queries. | 2025 | [Gemini 2.5 Technical Report](https://storage.googleapis.com/deepmind-media/gemini/gemini_2_5_report.pdf) |
| [**Anthropic**](./docs/anthropic-claude.md) | Models like Claude 3.7 Sonnet use **Extended Thinking** parameters to dial in reasoning budgets for deep logic. | 2025 | [Claude 3.7 Announcement](https://www.anthropic.com/news/claude-3-7-sonnet) |
| [**xAI**](./docs/xai-grok.md) | The **Grok 3** model features a dedicated "Think Mode" designed to dissect and plan complex tasks. | 2025 | [Grok 3 Blog](https://x.ai/blog/grok-3) |
